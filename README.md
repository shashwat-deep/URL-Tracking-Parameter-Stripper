# URL-Tracking-Parameter-Stripper
A lightweight utility that strips tracking parameters (like utm_*, fbclid, and gclid) from URLs and copies the clean link to your clipboard.

People should have the freedom to choose not to be "tracked", "monitored", or treated as a data asset, should they choose to. The ability to easily and quickly remove tracking parameters from URL while copying them, is a small step towards this belief.
